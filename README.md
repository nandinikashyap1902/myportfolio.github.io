# Portfolio Website 🌟

Welcome to my personal portfolio website! This project showcases my skills, projects, and professional background as a Web Developer. It is designed to be responsive, interactive, and visually engaging.

## 🚀 Live Demo
Visit the live site here: [Nandini's Portfolio](https://nandinikashyap1902.github.io/myportfolio.github.io/)

*(Note: If the link doesn't work directly, it might be hosted at the root domain or a different path depending on the GitHub Pages configuration.)*

## 📖 About
This portfolio serves as a central hub for my work, featuring:
- **About Me**: A brief introduction to my background and passion for web development.
- **Projects**: A showcase of my key projects, categorized into MERN Stack and AI/Generative AI ventures.
- **Skills**: A visual representation of my technical proficiency in various technologies.
- **Contact**: Easy ways to connect with me via LinkedIn, GitHub, or Email.

## ✨ Features
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
- **Interactive Animations**: Smooth scroll animations, fade-ins, and hover effects using CSS and JavaScript.
- **Dynamic Content**: Sections that highlight my latest work and technical capabilities.
- **Clean UI/UX**: A modern, easy-to-navigate interface with a focus on user experience.

## 🛠️ Technologies Used
- **HTML5**: For semantic structure and content.
- **CSS3**: For styling, layout (Flexbox/Grid), and animations (`animation.css`, `graphics.css`).
- **JavaScript**: For interactive elements like the mobile menu, scroll animations, and active navigation highlighting (`hello.js`).
- **Font Awesome**: For vector icons.

## 📂 Project Structure
```
├── images/           # Images and assets used in the site
├── animation.css     # CSS for scroll and element animations
├── graphics.css      # Additional graphical styles
├── hello.js          # Main JavaScript file for logic and interactivity
├── index.html        # Main HTML file
├── styles.css        # Core stylesheets
└── README.md         # Project documentation
```

## 🚀 How to Run Locally
1. **Clone the repository:**
   ```bash
   git clone https://github.com/nandinikashyap1902/myportfolio.github.io.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd myportfolio.github.io
   ```
3. **Open `index.html`:**
   Simply open the `index.html` file in your preferred web browser to view the site.
